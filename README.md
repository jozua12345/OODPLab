# OODP Lab
#### Working Repository for CZ2002 OODP

## Main Menu
- Press 1 for movie goer
- Press 2 for admin

## Movie goer Menu
- List all movies
- List top 5 movies
  - by sales
  - by rating
- Search movie
- View movie details
- Add movie review
- View movie showtimes
- Book ticket 
  - Purchase ticket
  
## Admin Menu
- Create movie
- Update movie
- Remove movie
- Update ticket price
- Create movie screening
- Update movie screening
- Remove movie screening
- Update holidays
